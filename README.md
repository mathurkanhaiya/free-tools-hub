# Free Utility Tools Hub

A fast, SEO-friendly utility website built with React, Vite, TypeScript, and Tailwind CSS.

## Included tool categories

- Calculators
- PDF Tools
- Image Tools

## Features

- Homepage with search, popular tools, recently used, and categories
- Dedicated tool pages with FAQs and structured content
- SEO landing pages for high-intent keyword clusters
- Static sitemap for crawlable page discovery
- Mobile-first responsive UI

## Routes

- `/` — homepage
- `/tool/:slug` — individual tool page
- `/category/:category` — category landing page
- `/seo/:slug` — SEO landing page

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
